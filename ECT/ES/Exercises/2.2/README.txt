--------------------------------------------------------------------------------
1) Create Modelsim project and add the SV file 'adder_4_bit.sv'
  or 'adder_c2_4_bit.sv'

2) Compile and launch the simulation selecting the module 'adder_4_bit'
  or 'adder_c2_4_bit'

--------------------------------------------------------------------------------
In case of 'adder_c2_4_bit', you can use the auxiliary .do files to open
the waveform and run the simulation, as shown during the lecture

3) Open the waveform:
  [Transcript]  >> do wave_adder_c2.do
  
4) Run the simulation:
  [Transcript]  >> do sim_adder_c2.do
